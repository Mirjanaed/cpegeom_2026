# OpenStreetMap (OSM)

Base de données géographique collaborative utilisée comme fond cartographique.

### Contenu
- Voirie
- Bâtiments
- Points d’intérêt
- Réseau cyclable et piéton

### Format
- .osm
- Export GeoJSON / Shapefile

### Utilisation
- Analyse réseau
- Calcul d’itinéraires
- Support cartographique
