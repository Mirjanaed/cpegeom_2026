# Présentation du projet

## Contexte

Les projets SIG développés au sein de TADAO ont pour objectif d’analyser, structurer et valoriser les données du réseau de transport.
Et également de faire des études de géomarketing.

## Objectifs

- Centraliser les données géographiques
- Améliorer l’analyse territoriale
- Optimiser les performances du réseau
- Développer des outils cartographiques dynamiques

## Environnement

- PostgreSQL / PostGIS
- Python
- Outils SIG (QGIS)
- Données ouvertes et internes
